# Folder generator for pentesting projects! Now on Br0mmie/foge

 - Quick and easy to use
 - Creates all folders needed
 - Can be scripted

## Installation

**Before doing any of this, make sure go is installed!!!**

```
wget https://github.com/Br0mmie/foge/releases/download/Alpha/install.sh
chmod +x install.sh
./install.sh
rm install.sh
```

## Usage

```
foge [PATH]
```

### Folder structure

The following stucture is generated when using `foge [PATH]`
